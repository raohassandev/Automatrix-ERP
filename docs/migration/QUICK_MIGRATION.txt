📋 QUICK STEPS (10 minutes)
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

1️⃣  UPLOAD OLD FILE
    • Upload Expanse.xlsx to Google Drive
    • Open with Google Sheets
    • Copy Sheet ID from URL

2️⃣  ADD MIGRATION SCRIPT
    • In NEW AutoMatrix: Extensions → Apps Script
    • Click + → New file → Name: MigrateOldData
    • Copy from: scripts/migrate-old-expanse.gs
    • Update OLD_SHEET_ID (line 21)
    • Update DEFAULT_USER_EMAIL (line 24): israrulhaq5@gmail.com
    • Save

3️⃣  RUN MIGRATION
    • Select: migrateAllOldData
    • Click Run ▶️
    • Check logs: View → Logs

4️⃣  VERIFY
    • Check Employees sheet - has your team?
    • Check Expenses - marked with [MIGRATED]?
    • Check Income - marked with [MIGRATED]?
    • Totals match old sheet?

✨ WHAT HAPPENS
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

Old Data is Copied with Special Marking:
  ✅ Expenses: "[MIGRATED] Original description"
  ✅ Income: "[MIGRATED] Original source"
  ✅ Status: Set to "Approved" (historical)
  ✅ Validation: Set to "Migrated"

Why Mark as [MIGRATED]?
  • Easy to identify old vs new data
  • Filter old data if needed
  • Keep historical records intact
  • Start fresh with new entries
