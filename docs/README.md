# Automatrix ERP - Project Documentation

This directory contains all documentation for the Automatrix ERP project, organized into thematic subdirectories.

## Contents

-   **`product/`**: Product specifications, requirements, workflows, role design, approval processes, and reporting needs.
-   **`tech/`**: Technical architecture, module designs, deployment strategies, Architectural Decision Records (ADRs), and system analysis.
-   **`migration/`**: Documentation related to data migration, including mappings from legacy Excel files to the new database schema, and detailed import steps.

## Usage

Refer to the appropriate subdirectory for specific documentation.
