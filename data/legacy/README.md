# Legacy Data

This directory contains legacy data files used as references for the Automatrix ERP project.

## Files

-   **`Automatrix_ERP.xlsx`**: The original Excel spreadsheet that served as the data source for the legacy Google Apps Script ERP. This file is considered a read-only reference for data migration and schema understanding.

## Usage

This data is primarily for historical reference and migration purposes. **Do not modify these files directly.**

For any data migration efforts to the new Next.js application, refer to the documentation in `docs/migration/`.
